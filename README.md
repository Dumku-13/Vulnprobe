# VulnProbe

Modular web vulnerability scanner built in Python.

## Modules
- scanner.py     — Port scanner with banner grabbing
- sqli_scanner.py — SQL injection detector (error-based, boolean-based, OR-based)

## Usage
python sqli_scanner.py
Enter URL, parameter, and session cookie when prompted.

## Tested on
- PortSwigger Web Security Academy labs
