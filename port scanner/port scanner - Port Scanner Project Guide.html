What is a socket (actual meaning)

A socket is just an endpoint for communication.

Think of it like this:

  • Your laptop → one endpoint

  • Server (like google) → another endpoint

  • Socket = the pipe between them

When you create a socket, you’re saying:

“I want to open a communication channel to another machine.”

What happens when you connect

When you run:

Python
s.connect_ex((target, port))

Under the hood:

  1. Your system sends a TCP SYN packet

  2. If port is open:

    • server replies SYN-ACK

    • connection established ✅

  3. If port is closed:

    • server replies RST

  4. If filtered/firewalled:

    • no reply → timeout

Your code doesn’t show packets, but that’s what’s happening.

What your socket object actually holds

When you do:

Python
s = socket.socket(...)

It creates a structure with:

  • protocol (TCP)

  • address family (IPv4)

  • connection state

  • file descriptor (OS-level handle)

It’s not just a variable. It’s tied to OS networking.

Lifecycle of a socket (important)

Every socket goes through:

  1. Created

  2. Connected (or failed)

  3. Closed

Once used, it’s basically “spent”.

Now your main question

Why create a new socket every loop?

Because a socket is stateful.

After you try one connection:

  • it’s either connected, failed, or timed out

  • it is no longer “fresh”

import socket
import threading

target = input("Enter target: ")
start_port = int(input("Start port: "))
end_port = int(input("End port: "))

print(f"\nScanning {target} from {start_port} to {end_port}...\n")

def scan_port(port):
    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    s.settimeout(1)

    result = s.connect_ex((target, port))

    if result == 0:
        print(f"Port {port} is OPEN")

    s.close()


threads = []

for port in range(start_port, end_port + 1):
    t = threading.Thread(target=scan_port, args=(port,))
    threads.append(t)
    t.start()

for t in threads:
    t.join()

print("\nScan complete.")

for t in threads:
    t.join()

end_time = time.time()

for port in sorted(open_ports):
